controller — сервлеты / обработчики REST
service — логика лотереи, пользователей, тиражей
repository — JDBC-запросы
model — POJO-сущности
dto — объекты для передачи по REST
security — аутентификация (например, Basic Auth или сессии)
util — генераторы комбинаций, валидаторы
